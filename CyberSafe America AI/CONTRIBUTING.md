# Contributing to NamKural

Thank you for your interest in contributing to NamKural — நம் குரல் (Our Voice). This is a public-interest initiative and every contribution, however small, helps protect Americans online.

## Ways to Contribute

- **Code** — PHP/Laravel, JavaScript, AI/ML, accessibility improvements
- **Content** — Educational materials, workshop curricula, multilingual translations
- **Design** — UI/UX improvements, accessibility audits
- **Research** — Cybersecurity data, community outreach findings
- **Documentation** — Guides, README improvements, whitepaper contributions

## Getting Started

1. Fork the repository
2. Create a feature branch from `develop`:
   ```bash
   git checkout -b feature/your-feature-name develop
   ```
3. Make your changes with clear, descriptive commits
4. Push your branch and open a Pull Request against `develop`

## Commit Message Format

We use [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add phishing quiz module for seniors
fix: correct score calculation in assessment engine
docs: update volunteer onboarding guide
style: improve mobile nav responsiveness
```

## Code Standards

- PHP: PSR-12 coding standard
- JavaScript: ES6+, no jQuery dependency
- HTML/CSS: Accessible markup, WCAG 2.1 AA minimum
- All user inputs must be sanitized and validated

## Pull Request Guidelines

- Keep PRs focused — one feature or fix per PR
- Include a clear description of what changed and why
- Reference any related issues
- Ensure the site still works on mobile before submitting

## Reporting Issues

Open a GitHub Issue with:
- A clear title and description
- Steps to reproduce (for bugs)
- Expected vs actual behavior
- Screenshots if relevant

## Code of Conduct

All contributors must follow our [Code of Conduct](CODE_OF_CONDUCT.md). We are committed to a welcoming, inclusive community.

## Questions?

Contact the founder via [namkural.org/contact](https://namkural.org/contact).
