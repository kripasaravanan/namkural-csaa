# NamKural Governance

NamKural is an independent public-interest initiative founded and maintained by Saravanan Kirubananthan (Kripa Saravanan).

## Decision Making
All major decisions rest with the founder. Community input is welcomed via GitHub Issues.

## Volunteers
Volunteers operate under the Code of Conduct. Cyber Ambassadors may represent NamKural with written acknowledgment from the founder.

## Financial
NamKural accepts no commercial sponsorships that influence program content.
